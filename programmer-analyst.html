Computer Programmer Analyst

I analyze requirements, design pragmatic solutions, and deliver maintainable software: C#/ASP.NET Core services, EF Core data access, SQL Server/MySQL data modeling, and Python/Flask for supporting tools. I pair clean architecture with clear documentation so changes are safe and fast.

  • Requirements & Systems Analysis
  • REST APIs (C# / ASP.NET Core)
  • EF Core · Migrations · Seeding
  • SQL: schema, queries, indexing, SPs
  • Auth: ASP.NET Identity (roles/claims)
  • Python/Flask utilities
  • Testing · Swagger
  • Docs & stakeholder comms

Where I add “senior” value

  • Translate business to design: partner with stakeholders, clarify scope/assumptions, write concise specs that developers can build from.
  • Model the data first: normalize schemas, design queries/indexes, and plan migrations so the system stays fast and correct as it grows.
  • Build stable APIs: DTOs, validation, error handling, and automated tests; Swagger for discoverability; auth/roles for safety.
  • Ship with confidence: small iterations, code reviews, and docs; prioritize observability and repeatable runbooks.

How I work

  1. Discover: partner with stakeholders, map workflows & data, write crisp acceptance criteria.
