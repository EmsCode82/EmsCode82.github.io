Assetgineer In Progress

AI-powered art generator and asset manager for game developers. Turns high-level concepts into consistent visual assets, then organizes them with versions, tags, and collections.

Highlights

  • Prompt engine understands races/biomes/architecture/styles; supports negative prompts and one-click randomize.
  • Image-to-image and in-painting for precise edits.
  • Prompt Analysis tree explains how prompts are built for transparent iteration.
  • Versioned library with tags/collections, bulk actions, and “load back into generator” for rapid iteration.
